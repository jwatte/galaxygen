Welcome to GitHub Pages.

After starting a Neptunes Pride 2 Triton game with absolutely zero planets visible from my starting cluster, I figured it should be possible to generate starting galaxies that are initially fair, yet still varied an interesting.

This site is an experiment in generating galaxy layouts for different number of players and different amounts of starting planets/resources, without running into significantly unbalanced positions.
